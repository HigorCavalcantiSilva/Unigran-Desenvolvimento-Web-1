Atividade 1

Em relação à linguagem JavaScript, assinale V para verdadeiro ou F para Falso.

a) (F) JavaScript é uma linguagem interpretada que requer compilação prévia.
b) (F) JavaScript precisa se comunicar com um servidor web para poder validar dados preenchidos em um formulário.
c) (V) Um script JavaScript pode reagir a eventos, como por exemplo: pode ser executado quando uma página termina de carregar no navegador web ou quando um usuário seleciona um elemento HTML.
d) (F) Assim como HTML, JavaScript não é case sensitive.
e) (F) Trechos de códigos JavaScript devem ser inseridos somente na seção HEAD de um documento HTML. A seção BODY não deve ser utilizada para este fim.
